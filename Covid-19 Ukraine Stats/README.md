# Covid-19 Ukraine Stats
This widget shows daily current statistics of COVID-19 in Ukraine.

## Features
Ukraine language.  
Displayed (for today):
- the number of identified patients today (updated in the morning around 9:00);
- the total number of patients;
- number of deaths;
- number of recovered.
